# LTI 1.3

## Definition

LTI (Learning Tools Interoperability) 1.3 is an IMS Global standard that enables secure LMS integration of external tools and content. It provides SSO (single sign-on), grade passback, and deep linking so learners access external tools seamlessly from within the LMS.

## When To Use It

- Use LTI when integrating third-party tools (video platforms, simulation tools, authoring tools, collaboration apps) into an LMS
- Use LTI 1.3 specifically for secure authentication (OAuth 2.0-based) and grade integration
- Reduces tool sprawl by providing a consistent integration interface

## Limitations

- Vendor support varies; not all tools implement LTI 1.3 fully
- Security and vendor risk reviews still needed per integration
- Does not replace xAPI for rich activity tracking — LTI handles integration, xAPI handles data
