# Information Architecture

## Summary

Information architecture (IA) is the structural design of shared information environments — how content is organized, labeled, and navigated so users can find what they need and understand where they are. In learning design, IA determines whether learners can navigate a course, find resources, and understand the program structure without confusion.

## Core Components

| Component | What it does | Example |
|---|---|---|
| **Organization** | How content is grouped and categorized | Modules by topic, by competency, by role, by difficulty |
| **Labeling** | How content is named | Module titles, menu labels, heading text — must be meaningful to the USER, not the designer |
| **Navigation** | How users move through the structure | Menu systems, breadcrumbs, next/previous, search, progress indicators |
| **Search** | How users find specific content | Search functionality, tagging, metadata |

## Key Principles

- **User mental models first** — organize by how the LEARNER thinks about the content, not how the SME or designer organizes it. Card sorting with actual users reveals this.
- **Consistent navigation** — the same type of content should always be in the same place. If resources are under "Materials" in Module 1, they should be under "Materials" in every module.
- **Progressive disclosure** — show overview first, detail on demand. Don't dump the entire course structure on page one.
- **Meaningful labels** — "Module 3" tells the learner nothing. "Handling Difficult Conversations" tells them exactly what's inside.
- **Findability > comprehensiveness** — it's better to have 20 well-organized resources than 200 that no one can find.

## Application to Learning Design

- **Course structure**: how modules, lessons, activities, and resources are organized and navigated within an LMS
- **Resource libraries**: how job aids, templates, and reference materials are organized for self-directed access
- **Knowledge bases**: how reference content is structured for findability (this knowledge base is itself an IA exercise)
- **Multi-program portfolios**: how an organization's full learning catalog is organized for learners to navigate
