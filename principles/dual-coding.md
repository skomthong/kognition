# Dual Coding Theory

## Definition

Dual coding theory (Paivio, 1971/1986) proposes that cognition operates through two distinct but interconnected systems: a **verbal system** (processing language, text, speech) and a **nonverbal/imagery system** (processing images, spatial information, sensory experience). Information encoded through both systems is more memorable than information encoded through one system alone.

## How It Works

- Verbal and imagery codes create **independent but connected** memory traces.
- When both systems encode the same information (e.g., a diagram with labels, a narrated animation), there are two retrieval routes instead of one — increasing recall probability.
- **Referential connections** between systems allow a word to activate an image and vice versa.
- Concrete concepts (which easily evoke images) are typically better remembered than abstract concepts (which rely more on verbal coding alone).

## Key Principles

- Pair verbal explanations with meaningful visuals — process diagrams, concept maps, annotated examples, timelines.
- Visuals must be **instructionally relevant** — decorative images increase extraneous load without adding a second encoding route.
- Dual coding is a foundational mechanism behind Mayer's [Multimedia Learning](multimedia-learning.md) principles (spatial contiguity, modality, multimedia effect).
- For adult learning: executive "one-pager" visual maps, process diagrams, and worked-example annotations leverage dual coding.

## When To Use It / When Not To

**Use when:** content can be meaningfully represented visually — processes, relationships, spatial layouts, hierarchies, comparisons. **Limitations:** not all content has natural visual representations. Forced or decorative visuals add load without benefit. The redundancy principle (Mayer) warns against duplicating narration as on-screen text.
