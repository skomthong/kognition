# Retention

## Definition

Retention is the process by which learned information or skills persist in long-term memory over time. It is the complement of forgetting — what is retained is what can later be retrieved and applied.

## Underlying Constructs

- [Cognitive Load](cognitive-load.md) — how effectively schemas are built during learning affects how durably they are retained
- [Narrative Cognition](narrative-cognition.md) — narrative structure provides retrieval scaffolds that aid retention
- Memory — retention is a property of the memory system, governed by encoding strength and storage consolidation
- Long-term memory — the store where retained knowledge resides

## How It Works

Retention depends on the quality of initial encoding (how deeply and how structurally information was processed), the number and spacing of subsequent retrievals, and the degree to which the learned material connects to existing schemas. Forgetting follows a decay curve (Ebbinghaus), but each successful retrieval flattens the curve and strengthens the memory trace.

Key factors:
- **Encoding depth** — elaborative processing produces more durable traces than shallow rehearsal.
- **Schema integration** — information connected to existing knowledge structures is retained longer.
- **Retrieval events** — each act of retrieval strengthens the memory, independent of restudying (the testing effect).
- **Spacing** — distributing practice over time produces stronger retention than massing it (the spacing effect).
- **Sleep and consolidation** — memory consolidation occurs during rest and sleep.

## Factors That Influence It

**Enhancers:**
- Retrieval practice (testing effect)
- Distributed/spaced practice
- Elaborative interrogation (asking "why" and "how")
- Interleaving related but distinct topics
- Meaningful context and authentic tasks
- Narrative embedding — stories provide retrieval scaffolds

**Inhibitors:**
- Massed practice (cramming)
- Passive review (rereading, highlighting)
- Cognitive overload during initial learning
- Lack of retrieval opportunities after learning
- Weak transfer climate (no reinforcement in the work environment)

## Methods That Leverage This

- [Retrieval Practice](../methods/retrieval-practice.md) — testing as learning, not only assessment
- [Gamification](../techniques/narrative/gamification.md) — streaks and progress tracking sustain the repetition needed for retention
- [Microlearning](../methods/microlearning.md) — short spaced segments enable distributed practice within constrained schedules
- [Scenario-Based Learning](../techniques/narrative/scenario-based-learning.md) — contextually embedded decisions are retained more durably than decontextualized rules
- Distributed practice / spaced repetition — scheduling review at expanding intervals
- Elaborative interrogation — generating explanations during study
- Interleaving — mixing related topics to strengthen discrimination
