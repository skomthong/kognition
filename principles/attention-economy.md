# Attention Economy

## Definition

The attention economy (Simon, 1971; Davenport & Beck, 2001) recognizes that in an information-rich world, attention — not information — is the scarce resource. Every learning experience competes with email, Slack, social media, meetings, and the learner's own racing thoughts. Designing for the attention economy means designing for REAL humans in REAL environments, not idealized learners in quiet rooms.

## How It Works

- **Attention is finite and depletable** — every decision, notification, and context switch consumes attentional resources
- **Competition is constant** — your e-learning module competes with the learner's inbox, phone, and to-do list
- **Switching cost is real** — every interruption requires 15-25 minutes to fully re-engage with deep work (Mark et al.)
- **Voluntary attention requires motivation** — without clear relevance and value ([expectancy-value](../foundations/expectancy-value.md)), learners allocate attention elsewhere

## Key Principles

- **Earn attention, don't assume it** — the first 30 seconds determine whether the learner stays. [Engagement techniques](../techniques/narrative/engagement-techniques.md) matter because attention is voluntary.
- **Protect attention once earned** — reduce extraneous load, minimize UI friction, eliminate notifications during learning, design for uninterrupted focus periods.
- **Design for fragmented attention** — adults learn in stolen moments. [Microlearning](../techniques/instructional/microlearning.md), mobile-first design, and resumable content respect how attention actually works.
- **Make the learning worth more than the alternative** — the real competition isn't boredom; it's everything else the learner could be doing with that time. Relevance and immediate applicability ([andragogy](andragogy.md)) win the attention allocation decision.
- **Reduce decision fatigue** — every choice consumes attention. [Default pathways](../techniques/instructional/choice-architecture.md) reduce the number of decisions required.
- **Notification design matters** — learning nudges compete with 100+ daily notifications. Make them valuable, timely, and respectful.

## Application to Learning Design

| Design choice | Attention-economy logic |
|---|---|
| Short segments with clear objectives | Respects fragmented attention |
| Strong opening hooks | Earns voluntary attention |
| Minimal UI, consistent navigation | Reduces attentional cost of interface |
| Default pathways with opt-out | Reduces decision fatigue |
| Just-in-time delivery | Arrives when attention is available and motivation is high |
| Clear "what's in it for me" | Wins the allocation competition |
