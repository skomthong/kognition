# Change Management Models

## Definition

Change management provides structured approaches for transitioning individuals, teams, and organizations from a current state to a desired state. In learning design, change management is essential when introducing new programs, platforms, pedagogies, or organizational learning cultures — because training that ignores organizational readiness fails at transfer.

## Key Models

### Kotter's 8-Step Model
| Step | Action | L&D application |
|---|---|---|
| 1. Create urgency | Show why change is needed NOW | Present data on current program failures (retention, transfer, ROI) |
| 2. Form a coalition | Assemble influential supporters | Identify champion executives, respected faculty, early-adopter managers |
| 3. Create a vision | Define the desired future state | "All programs will include spaced retrieval and transfer measurement" |
| 4. Communicate the vision | Repeat through multiple channels | Town halls, demos, pilot success stories, one-on-ones |
| 5. Empower action | Remove barriers | Fix LMS friction, provide time allocation, train facilitators |
| 6. Create quick wins | Demonstrate early success | Pilot one program, show results, publicize |
| 7. Build on change | Use momentum to tackle bigger changes | Expand from pilot to portfolio |
| 8. Anchor in culture | Make it "how we do things" | Update standards, hiring criteria, incentive structures |

### ADKAR (Prosci)
Individual-level change model:
- **A**wareness — of the need for change
- **D**esire — to participate and support
- **K**nowledge — of how to change
- **A**bility — to implement the change
- **R**einforcement — to sustain the change

Use ADKAR as a diagnostic: if adoption is stalling, which element is missing?

## Key Principles

- **Training is not change management** — training addresses Knowledge and (partially) Ability. Without Awareness, Desire, and Reinforcement, training alone fails.
- **Manager involvement is the #1 predictor** — if managers don't support, model, and reinforce the change, it dies regardless of training quality.
- **Quick wins build credibility** — pilot small, show results, then scale. Trying to change everything at once produces resistance and fatigue.
- **Resistance is information** — resistance signals unaddressed concerns (usually about Awareness or Desire). Investigate, don't override.
