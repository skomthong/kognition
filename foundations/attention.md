# Attention and Perception

## Definition

Attention is the cognitive system that selects information for further processing. Broadbent's filter theory formalizes selective attention within information-processing. Posner & Petersen describe attention as a system with separable functions: alerting, orienting, and executive control. Raymond, Shapiro & Arnell characterize the attentional blink — limits in rapid serial processing after detecting a target.

Gestalt perceptual organization (Wertheimer) describes how humans group visual elements: proximity, similarity, good form/Pragnanz, continuity, closure, figure-ground.

## How It Works

Attention gates what enters working memory from sensory memory. If attention does not select information, encoding cannot happen. In learning environments, two types of attention interact:

- **Goal-directed (top-down)**: learner intentionally focuses on task-relevant information
- **Stimulus-driven (bottom-up)**: salient features capture attention regardless of goals (animations, pop-ups, novel elements)

Design must support goal-directed attention and minimize unwanted stimulus-driven capture.

## Key Principles

- Adults in workplaces face interruption and multitasking; design for short segments and clear cues to reduce attentional switching costs.
- Apply Gestalt grouping intentionally in slides, dashboards, and interactive modules so learners perceive structure without extra cognitive integration effort.
- One primary focus per screen; avoid competing animations.
- **Signaling principle** ([Multimedia Learning / Mayer](multimedia-learning.md)): use arrows, highlights, headings, and verbal cueing to direct attention to essential elements without adding content.
- Salience budgeting: attention capture is not the same as learning. "Engaging visuals" can increase extraneous load if they don't support the learning goal.
