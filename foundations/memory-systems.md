# Memory Systems

## Definition

Memory is the set of cognitive systems that encode, store, and retrieve information. The multi-store model (Atkinson & Shiffrin, 1968) proposes structural stores — sensory, short-term, and long-term — with control processes governing flow between them. This architecture is the foundation for most instructional design decisions about sequencing, load, practice, and assessment.

## How It Works

**Sensory memory** holds brief impressions from the senses. Iconic memory (visual, ~250ms; Sperling) and echoic memory (auditory, ~2-4s; Neisser/Crowder & Morton) gate what enters further processing. Only attended information transfers to working memory.

**Working memory** (Baddeley & Hitch, 1974) is a multi-component system — central executive, phonological loop, visuospatial sketchpad, and episodic buffer — that holds and manipulates a small amount of information during complex cognition. Capacity is closer to 3-5 chunks (Cowan's "magical number 4") than the classic "7±2" (Miller). This bottleneck is the central constraint that cognitive load theory addresses.

**Long-term memory** has effectively unlimited capacity and stores organized knowledge structures:
- **Declarative memory** (Squire): episodic (personal events in time/space; Tulving) and semantic (organized knowledge)
- **Nondeclarative memory** (Squire): procedural skills, habits, priming, conditioning

**Intermediate-term memory** is discussed in neuroscience as a bridging state between working and long-term memory, but it is not a standard instructional design category.

## Key Principles

- Adults bring dense prior knowledge structures (schemas). Design should activate relevant schemas early and avoid forcing novices and experts through identical guidance (expertise reversal).
- Treat memory as retrieval-dependent performance under context: retrieval cues should be aligned with target performance settings (encoding specificity; Tulving & Thomson).
- For workplace competence, prioritize proceduralization (procedural/nondeclarative memory) and retrieval fluency under realistic cues, not only declarative recall.
- Working-memory limits constrain what can be processed at any moment — this is why [Cognitive Load](../principles/cognitive-load.md) management is non-negotiable.
