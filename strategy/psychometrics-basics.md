# Psychometrics Basics

## Definition

Psychometrics is the science of measuring psychological constructs — knowledge, skills, abilities, attitudes — through standardized instruments. For learning designers, psychometric literacy means understanding how to build fair, reliable, valid assessments and how to interpret their results correctly.

## Core Concepts

### Classical Test Theory (CTT)
- **Observed score = True score + Error**
- Every measurement contains some error (random variation in performance, guessing, environmental factors)
- Reliability is the proportion of observed score variance that is true score variance (reliability = 1 - error proportion)

### Item Response Theory (IRT) — advanced
- Models the probability of a correct response as a function of person ability AND item characteristics (difficulty, discrimination)
- Enables: computer adaptive testing, item banking, equating across test forms
- Relevant for large-scale assessment programs and certification

### Item Analysis Metrics

| Metric | What it tells you | Target range |
|---|---|---|
| **Difficulty (p-value)** | Proportion correct | 0.3 - 0.8 for good discrimination |
| **Discrimination** | Does the item separate high and low performers? | > 0.20 is acceptable; > 0.30 is good |
| **Point-biserial correlation** | Correlation between item score and total score | Positive and significant |
| **Distractor analysis** | Are wrong options attracting responses? | Each distractor should attract some responses |

### Differential Item Functioning (DIF)
- An item functions differently across demographic groups (gender, ethnicity, language) beyond what ability differences explain
- DIF ≠ bias (DIF is statistical; bias is a judgment about fairness), but DIF flags items for review
- Essential for high-stakes and credentialing assessments

### Standard Setting
- How do you determine the "pass" score? Not arbitrary — use structured methods:
  - **Angoff method**: experts estimate the probability that a minimally competent person would answer each item correctly
  - **Bookmark method**: experts place a "bookmark" in an ordered item booklet where the minimally competent boundary falls
- Standard setting is a judgment process, not a mathematical one — it requires defensible rationale

## Key Principles

- **Every assessment IS a measurement instrument** — treat it with measurement discipline, even for formative use
- **Item analysis is quality control** — after every administration, analyze and improve
- **Fairness requires evidence** — DIF analysis, accommodation policies, and bias review are not optional for consequential assessments
- **Cut scores require justification** — "70% to pass" is arbitrary without a standard-setting process
