# Speechwriting

## Summary

Speechwriting is writing for the ear, not the eye. Every principle changes: sentences are shorter, structure is more repetitive, rhythm matters more, and the emotional arc must be designed because listeners can't reread. Great speeches work because they sound inevitable, not because they read well on paper.

## How Speeches Differ from Written Text

| Written text | Spoken text |
|---|---|
| Reader controls pace (can reread) | Listener can't go back — clarity must be immediate |
| Complexity is manageable (reader can pause) | Complexity is lost — keep it simple |
| Visual structure (headings, bullets) aids navigation | Verbal structure (signposting, repetition) must do this work |
| Precision is everything | Rhythm and emphasis carry meaning alongside words |
| Can be dense | Must breathe — pauses, short sentences, white space in time |

## Speech Structure

### Opening (hook the audience)
- Start with a story, a question, a surprising fact, or a bold statement
- Do NOT start with "Thank you for having me" or "Today I'm going to talk about..."
- The first 30 seconds determine whether the audience listens

### Body (3 points maximum)
- Rule of three: audiences remember 3 things, not 7
- Each point: claim → evidence/story → implication → transition
- Repeat the key message — listeners need to hear it multiple times
- Use anaphora and parallel structure to create rhythm and emphasis

### Closing (land the message)
- Return to the opening (callback) — creates closure
- End with a call to action or a resonant image
- The last sentence should be memorizable

## Rhythm and Cadence

- **Short sentences for emphasis.** Long sentences for explanation and nuance, building toward a point that lands—
- **Hard.** (Like that.)
- Vary sentence length deliberately ([Sentence Control](sentence-control.md))
- Use parallel structure: "We came. We saw. We conquered."
- Write in phrases, not paragraphs — mark where the speaker should breathe, pause, emphasize

## Craft Exemplars

| Speaker | Signature technique |
|---|---|
| **MLK** | Anaphora ("I have a dream..."), building rhythm, biblical cadence, imagery |
| **Obama** | Long-short sentence variation, "false choice" reframing, narrative callbacks |
| **JFK** | Chiasmus ("Ask not..."), antithesis, compressed power |
| **Churchill** | Short Anglo-Saxon words under stress, building lists, emotional climax |

## How To Practice

1. Write a 2-minute speech (roughly 300 words). Read it aloud. If you stumble, rewrite the stumble points.
2. Record yourself delivering it. Listen back — where does the energy drop? That's where the writing needs rhythm.
3. Analyze one famous speech: mark the rhetorical devices, the sentence length variation, and the emotional arc.
