# Logical Sequencing

## Summary

Logical sequencing is choosing the right ORDER for presenting ideas. The sequence determines whether the reader builds understanding progressively or gets lost in a jumble of disconnected points. Different content types have natural orderings.

## Sequencing Patterns

| Pattern | Logic | Best for |
|---|---|---|
| **Chronological** | Time order — first, then, next, finally | Processes, procedures, narratives, historical development |
| **Causal** | Cause → effect, or effect → cause | Explaining why something happened, problem analysis |
| **Problem–Solution** | Present problem, then propose solution | Recommendations, proposals, case studies |
| **General → Specific** | Start with the big picture, then zoom in | Introductions, overviews, progressive detail |
| **Specific → General** | Start with details, build to a conclusion | Inductive arguments, research findings leading to theory |
| **Compare–Contrast** | Alternate between two things, or block one then the other | Evaluations, alternative analysis, decision support |
| **Priority / importance** | Most important first, or build to the most important | Recommendations (important first), persuasion (build to climax) |
| **Known → Unknown** | Start with what the reader knows, bridge to new | Teaching, andragogy (activate prior knowledge first) |
| **Simple → Complex** | Foundation concepts before advanced | Instructional sequencing, progressive curriculum |

## Section Purpose Alignment

Every section in a document should have a clear PURPOSE. Ask: "What job does this section do for the reader?"

| Section purpose | What it does |
|---|---|
| **Orient** | Establish context, activate prior knowledge |
| **Inform** | Present new information |
| **Argue** | Make and support a claim |
| **Instruct** | Tell the reader what to do |
| **Summarize** | Consolidate what was covered |
| **Transition** | Bridge from one topic to the next |

If a section doesn't serve one of these purposes, it probably shouldn't exist.

## How To Practice

1. Before writing, list the points you need to make. Then sequence them using one of the patterns above. Justify your choice.
2. Take a document and label each section's purpose. If you can't label it, the section lacks focus.
3. Try resequencing the same content in a different order. Notice what changes in reader experience.
