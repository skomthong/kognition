# Media Scripting

## Summary

Media scripting is writing the spoken narration, on-screen text, and visual directions for video, audio, and interactive media. It applies multimedia learning principles at the sentence level — every word and visual choice affects cognitive load, attention, and retention.

## Script Formats

### Two-column video script
| Audio (what is heard) | Visual (what is seen) |
|---|---|
| "When a customer escalates, your first move is to acknowledge their emotion." | Close-up of facilitator. Text overlay: "Step 1: Acknowledge" |
| "Watch how this plays out in a real scenario." | Cut to scenario video clip |

### Audio/podcast script
Single column with speaker labels, timing notes, and production cues:
```
[INTRO MUSIC — 5 sec, fade under]
HOST: Welcome back. Today we're tackling the hardest part of feedback — what to say when someone's not meeting expectations.
[PAUSE 1 sec]
HOST: Let's start with a scenario...
```

## Key Principles from CTML

- **Write for the ear, not the eye**: narration should be conversational, not academic. Personalization principle (Mayer) — conversational style improves learning.
- **Don't narrate what's on screen**: if a diagram is labeled, don't read the labels aloud (redundancy principle). Narrate the insight, not the text.
- **Signal structure verbally**: "There are three steps. First... Second... Third..." Verbal signaling helps working memory organize incoming information.
- **Segment**: break narration into short chunks with pauses. Match audio segments to visual changes.
- **Script silence**: deliberate pauses give working memory processing time. Don't fill every second with narration.

## Common Mistakes

- Writing in "document voice" instead of spoken voice — too formal, too dense, too long
- Reading slides aloud (redundancy overload)
- No visual plan — the script describes words but not what the learner sees
- Too much information per segment — working memory overload
- No retrieval prompts — passive narration without embedded questions

## How To Practice This Skill

1. Write a 2-minute narration script for a concept you know well. Read it aloud — does it sound natural?
2. Convert an existing text-heavy slide deck into a two-column script. What do you cut? What becomes visual?
3. Record yourself delivering the script. Listen back. Where do you stumble? That's where the writing needs fixing.
