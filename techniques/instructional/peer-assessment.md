# Peer Assessment

## Summary

Peer assessment is structured evaluation where learners assess each other's work using explicit criteria. Research shows the ASSESSOR often learns more than the person being assessed — the act of evaluating forces analytical processing, calibration against standards, and metacognitive judgment.

## How To Apply It

### Structure is non-negotiable
Unstructured "give each other feedback" produces surface-level comments. Structure with:
- **Rubric or criteria** — explicit, shared, understood before the task
- **Focused prompts** — "Does the argument include evidence for each claim? If not, identify which claims lack support." NOT "What do you think?"
- **Multiple rounds** — first peer feedback → revision → second feedback (iterative improvement)

### Assessment vs. feedback
| Mode | Purpose | Grading? |
|---|---|---|
| **Peer feedback** | Improve the work through suggestions | No grades — developmental only |
| **Peer assessment** | Evaluate the work against criteria | Contributes to grades (use with calibration) |
| **Self-assessment** | Learner evaluates own work against criteria | Usually paired with peer/instructor assessment for calibration |

## Key Principles

- **The reviewer learns more than the reviewed** — evaluating others' work develops analytical judgment, standard internalization, and metacognitive calibration
- **Calibration is essential** — before assessing peers, learners must practice applying criteria to example work and compare judgments. Without calibration, peer assessment is unreliable.
- **Anonymity has trade-offs** — anonymous peer review reduces social pressure but also reduces accountability and developmental relationship
- **Psychological safety determines quality** — if learners fear retaliation or social consequences, they'll give bland feedback. Establish norms explicitly.
- **Self-assessment alone is unreliable** — adults consistently over- or under-estimate their own performance. Pair with peer and instructor feedback for calibration.
