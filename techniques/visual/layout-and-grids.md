# Layout and Grid Systems

## Summary

A grid system is an invisible structure that aligns elements consistently across pages and screens. It creates visual order without the viewer noticing it — content looks organized, professional, and easy to scan because everything is aligned to the same underlying structure.

## How Grids Work

- **Columns**: the vertical divisions of the page (4-column, 8-column, 12-column are common)
- **Gutters**: the space between columns
- **Margins**: the space around the edge
- **Rows**: optional horizontal divisions for vertical rhythm

Content elements snap to column edges. This creates automatic alignment and consistent spacing.

## Key Principles

- **Consistency across pages/screens**: every page in a course should follow the same grid. The learner develops spatial expectations (content is always here, navigation is always there).
- **Responsive grids**: on smaller screens, columns stack. Design for the smallest screen first, then expand.
- **Consistent spacing**: use a base unit (8px is common) and multiply for all spacing decisions — 8, 16, 24, 32, 48. This creates rhythm.
- **Templates, not one-offs**: define 3-5 layout templates for common content types (text + image, full-width media, two-column comparison, quiz screen) and reuse them. Consistency reduces the learner's cognitive overhead for navigating the interface.

## Common Mistakes

- **No grid at all** — elements placed by eyeball, slightly off from each other. Looks amateurish.
- **Breaking the grid randomly** — deliberate grid-breaking for emphasis works; accidental grid-breaking looks like mistakes.
- **Too many layout variations** — every screen looks different, forcing the learner to re-orient each time.
- **Ignoring mobile** — content designed for desktop that breaks on mobile (text overflows, images don't scale, touch targets too small).

## How To Practice

1. Turn on the grid overlay in your design tool (PowerPoint: guides, Figma: layout grid). Snap everything to it.
2. Define a 3-template system for a course: content screen, interaction screen, summary screen. Design all content using only these three.
3. Print two pages of a learning module and overlay them. Do elements align? If not, you need a grid.
