# Gestalt Principles in Design

## Summary

Gestalt principles (Wertheimer, 1923) describe how humans naturally organize visual elements into groups, patterns, and wholes. In learning design, applying Gestalt deliberately reduces the cognitive effort needed to perceive structure — the brain does the organization work automatically, freeing working memory for content.

## The Principles Applied

| Principle | What the brain does | Design application |
|---|---|---|
| **Proximity** | Groups nearby elements together | Place related items close; separate unrelated items with space |
| **Similarity** | Groups elements that look alike (color, shape, size) | Use consistent styling for similar elements (all key terms in bold, all examples in the same format) |
| **Continuity** | Follows lines and curves smoothly | Align elements along invisible lines; use arrows and flow lines to guide the eye |
| **Closure** | Completes incomplete shapes | Use partial borders, implied containers; the brain fills in gaps |
| **Figure-ground** | Separates focal element from background | Ensure clear contrast between content and background; no ambiguity |
| **Common region** | Groups elements within a shared boundary | Use boxes, cards, or shaded areas to group related content |
| **Common fate** | Groups elements that move/change together | In animations: elements that animate together are perceived as a unit |

## Application to Learning Materials

- **Slides**: group related bullet points with proximity; separate sections with white space; use common region (boxes) for key takeaways
- **Dashboards**: proximity groups related data; similarity distinguishes categories; figure-ground separates active from background data
- **E-learning screens**: consistent visual treatment for interactions, feedback, and content; proximity between questions and their context
- **Diagrams**: continuity guides the eye through a process; closure allows simplified representations; similarity codes categories

## How To Practice

1. Take a cluttered learning screen and identify Gestalt violations (related items far apart, unrelated items close, inconsistent styling). Redesign.
2. Design a diagram using only proximity and similarity — no labels. Can a viewer understand the groupings?
